# Dental AI Platform - PWA Demo

Умная платформа для планирования стоматологического лечения с ИИ

## Возможности

### Для пациентов
- Загрузка КТ-снимков
- Автоматический план лечения от ИИ
- Получение предложений от клиник
- Выбор клиники по критериям
- Отслеживание статуса лечения

### Для клиник
- Получение заявок от пациентов
- Автоматический расчёт стоимости
- Управление лидами
- Аналитика и отчёты

### Для страховых компаний
- Отслеживание лечения пациентов
- Контроль стоимости услуг
- Андеррайтинг

### Для департамента здравоохранения
- Мониторинг рынка
- Агрегированная статистика
- Контроль жалоб
